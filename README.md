# 🚀 Code-Stash

A sleek, cloud-based code snippet manager built with **React**, **Tailwind CSS**, and **Firebase**.

## ✨ Features
- **Google Authentication:** Secure login for every user.
- **Realtime Database:** Your snippets are saved in the cloud and synced across devices.
- **Syntax Highlighting:** Beautiful code display using Prism.js.
- **Search & Filter:** Find your snippets by title or language in seconds.
- **Responsive Design:** Works perfectly on mobile and desktop.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, React Router
- **Backend:** Firebase (Auth & Realtime Database)
- **Icons:** React Icons

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/rajnaman330/code-stash.git`
2. Install dependencies: `npm install`
3. Add your Firebase config in `src/firebase.js`
4. Start development server: `npm run dev`
